# Personal Portfolio Website

This repository houses the source code for my personal portfolio website: [javierchuaby.github.io](https://javierchuaby.github.io).

## Tech Stack

The website is a modern, responsive, and performance-optimized static single-page application built using:

- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 & custom properties
- **Formatting/Linting**: Biome
- **Testing**: Vitest with React Testing Library

## Development

To run the development server locally:

```bash
npm run dev
```

To run the test suites:

```bash
npm test
```

To run the linter:

```bash
npm run lint
```

To compile a static export:

```bash
npm run build
```
